1) 
Get chrome / chromium browser. 
Change CHROME_BINARY_PATH to your location of chrome.exe. 

2) 
Get Chrome driver, from:
https://googlechromelabs.github.io/chrome-for-testing/#stable

Set CHROMEDRIVER_PATH to your location of chrome driver.


3) 
# Run scrapping
```bash
cd zprp/scraper
python -m venv .venv
.venv/Scripts/activate
pip install -r requirements.txt
python main.py
```
