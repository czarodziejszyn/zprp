1. Move .json file generated by running /scraper/main.py to geocoding directory.

# Set up Nominativ server:
2. log in to the docker in terminal: (you need to have docker account) 
docker login


3. Run server:
docker run -d -p 6080:8080 -e NOMINATIM_PBF_URL="https://download.bbbike.org/osm/bbbike/Warsaw/Warsaw.osm.pbf" -e POSTGRES_PASSWORD="haslo" --name nominatim-warsaw peterevans/nominatim:latest

Observe the logs. Setup should take around 20-30min. 
docker logs -f nominatim-warsaw


After it is finished, you have Nominativ server ready at:
http://localhost:6080/ 


4. To stop / start anew:
docker stop nominatim-warsaw
docker start nominatim-warsaw





# Run gecoding. This will create .json file with geocoded offers in /geocoding. This file can be used in backend, during db data import. 
docker compose build --no-cache
docker compose up
